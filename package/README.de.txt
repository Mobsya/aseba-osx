                      ASEBA LIES MICH
                     -------------

Executables können direkt gestartet werden. Sie sind:
    asebastudio             IDE
    asebaswitch             software switch
    asebadump               dump ASEBA packets
    asebacmd                debug control of nodes (bootloader, ...)
    asebachallenge          Programmier Spiel mehrere Spieler in einer einzigen Arena
    asebaplayground         Programmier Spiel, mehrere Roboter für einen Spieler,
                             kompatibel mit physischen E-Puck.
