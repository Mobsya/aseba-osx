                      ASEBA README
                     -------------

Les programmes suivants peuvent être démarrés directement:
    asebastudio             Environnement de programmation
    asebaswitch             Commutateur logiciel
    asebadump               Dump des paquets ASEBA
    asebacmd                Contrôle (de debug) des noeuds (bootloader, etc.)
    asebachallenge          Jeu de programmation, multi-joueurs, dans une seule arène
    asebaplayground         Jeu de programmation, multi-robots pour un joueur
                            compatible avec le robot e-puck réel.
