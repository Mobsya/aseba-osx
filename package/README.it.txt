                      ASEBA README
                     -------------

I seguenti programmi possono essere avviati direttamente:
    asebastudio             Ambiente di programmazione
    asebaswitch             Softswitch
    asebadump               Dump pacchetti ASEBA
    asebacmd                Controllo (di debug) dei nodi (bootloader, etc.)
    asebachallenge          Gioco di programmazione, multi-player, in un'arena
    asebaplayground         Gioco di programmazione, multi-robot per un giocatore 
                            compatibile con l'attuale robot e-puck.
